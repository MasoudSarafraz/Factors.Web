# سیستم مدیریت فاکتور - Factors.Web

سیستم مدیریت فاکتور فارسی با ASP.NET Core 8 MVC

## ویژگی‌ها

### 📋 مدیریت داده‌ها
- **دسته‌بندی محصولات**: ایجاد، ویرایش، حذف دسته‌بندی‌ها
- **محصولات**: CRUD کامل با مدیریت قیمت‌ها و تاریخچه قیمت
- **اشخاص**: مدیریت مشتریان حقیقی و حقوقی
- **بسته‌ها**: تعریف بسته‌های محصولات با آیتم‌های مختلف
- **فاکتورها**: صدور فاکتور با انتخاب محصول/بسته، محاسبه خودکار جمع کل

### 🔐 امنیت و دسترسی
- **سیستم احراز هویت**: ورود با نام کاربری و رمز عبور
- **مدیریت نقش‌ها**: سه نقش پیش‌فرض (Admin, Manager, User)
- **کنترل دسترسی**: محدودیت دسترسی بر اساس نقش
- **قفل حساب**: بعد از 5 تلاش ناموفق
- **مدیریت کاربران**: ایجاد، ویرایش، فعال/غیرفعال، حذف کاربران (فقط Admin)
- **تغییر رمز عبور**: توسط مدیر

### 📊 گزارشات
- **گزارش فاکتورها**: با فیلتر تاریخ و مشتری
- **گزارش فروش**: خلاصه فروش در بازه زمانی
- **گزارش محصولات**: لیست محصولات با فیلتر دسته‌بندی
- **خروجی PDF**: تمام گزارشات قابل چاپ

### 📱 رابط کاربری
- **Bootstrap 5 RTL**: طراحی فارسی و راست‌به‌چپ
- **jQuery AJAX**: عملیات بدون رفرش صفحه
- **داشبورد**: آمار کلی، آخرین فاکتورها، پرفروش‌ترین محصولات
- **مودال‌ها**: فرم‌های ایجاد و ویرایش در مودال

## پیش‌نیازها

- .NET 8 SDK
- Visual Studio 2022 یا VS Code

## نصب و راه‌اندازی

### 1. کلون یا دانلود پروژه

### 2. بازیابی پکیج‌ها
```bash
cd Factors.Web
dotnet restore
```

### 3. ایجاد دیتابیس (خودکار)
دیتابیس SQLite و داده‌های اولیه هنگام اولین اجرا خودکار ایجاد می‌شوند.

### 4. اجرای پروژه
```bash
dotnet run
```

یا در Visual Studio، پروژه را باز کرده و F5 بزنید.

### 5. ورود با حساب پیش‌فرض

| نقش | نام کاربری | رمز عبور |
|------|-----------|----------|
| Admin | admin | Admin@123 |
| Manager | manager | Manager@123 |

## سطوح دسترسی

### Admin (مدیر سیستم)
- دسترسی کامل به تمام بخش‌ها
- مدیریت کاربران و نقش‌ها
- حذف دسته‌بندی، محصول، شخص، فاکتور
- مشاهده و تولید گزارشات

### Manager (مدیر فروش)
- مدیریت فاکتورها (ایجاد، مشاهده، حذف)
- مشاهده و تولید گزارشات
- ویرایش محصولات و بسته‌ها

### User (کاربر عادی)
- مشاهده دسته‌بندی‌ها، محصولات، اشخاص
- ایجاد و مشاهده فاکتورها

## تکنولوژی‌ها

- **ASP.NET Core 8** MVC
- **Entity Framework Core 8** (SQLite)
- **ASP.NET Core Identity** (احراز هویت)
- **QuestPDF** (تولید PDF)
- **Bootstrap 5 RTL**
- **jQuery 3.7**
- **Bootstrap Icons**

## نکات مهم

1. دیتابیس SQLite به صورت فایل `factors.db` در پوشه اجرایی ایجاد می‌شود
2. برای تغییر رمز عبور پیش‌فرض، بعد از اولین ورود از بخش مدیریت کاربران استفاده کنید
3. تاریخ‌ها به صورت شمسی نمایش داده می‌شوند
4. تمام فرم‌ها دارای اعتبارسنجی سمت کلاینت و سرور هستند
